# Contributing to WindowsGSM.Factorio

Thanks for your interest in improving this plugin!

## How to Contribute

### 1. Fork the Repository
Create your own fork and clone it locally.

### 2. Make Your Changes
Keep code clean, documented, and consistent with the existing style.

### 3. Test Your Changes
- Install the plugin in WindowsGSM
- Verify SteamCMD installation works
- Confirm the server starts and loads a save

### 4. Submit a Pull Request
Describe:
- What you changed
- Why it improves the plugin
- Any testing you performed

## Code Style
- Use clear, descriptive variable names
- Avoid hardcoded paths
- Keep logic modular and readable

## Reporting Issues
Open an issue with:
- WindowsGSM version
- Plugin version
- Reproduction steps
- Logs or screenshots if possible
